Quadratic Equations

Reminder about quadratic equation

Quadratic equation has a form of: \[ a{x^2}+b{x}+c=0 \]

And represents a parabola

It has two solitions called ROOTS

If parabola crosses the "Y" axis - it has 2 real roots - like in figure 1. If parabola doesn't cross the "Y" axis - it has no real roots - like in figure 2, or it can have only one solution like in figure 3.

To get the solution use the formula: \[ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]

Note 1: if \( b^2 - 4ac < 0\) - there is no real solution

Note 2: if \( b^2 - 4ac = 0\) - there is 1 solution

It's quite simple
